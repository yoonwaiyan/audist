# Product Overview

## Project Purpose
Audist is an AI-powered meeting recorder application that captures audio from meetings, performs speaker recognition, and generates intelligent summaries. The application provides a desktop solution for recording, transcribing, and analyzing meeting conversations.

## Key Features
- **Audio Recording**: Capture system audio from meeting platforms (Zoom, Google Meet) and microphone input
- **Speaker Recognition**: Identify and differentiate between multiple speakers in a conversation
- **AI-Powered Summaries**: Generate smart summaries of meeting content using AI
- **Waveform Visualization**: Display audio waveforms for visual feedback during recording
- **Cross-Platform Desktop App**: Built with Electron for macOS and Windows support
- **Settings Management**: Configure recording preferences and audio sources

## Target Users
- Professionals who attend frequent virtual meetings
- Teams needing accurate meeting documentation
- Users requiring speaker-attributed transcriptions
- Anyone seeking automated meeting summaries

## Use Cases
- Recording and transcribing team meetings
- Capturing client calls with speaker identification
- Generating meeting minutes automatically
- Reviewing meeting content with visual audio feedback
- Archiving important conversations with AI-generated summaries
